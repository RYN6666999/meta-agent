📅 Google 日曆連結

📊 Google 試算表（日報同步）

      
授權後可將日報表一鍵同步到指定的 Google 試算表，供主管檢視。

      

        

          
          
          未連結
        

        

          
          
          
        

        

          同步後會在試算表新增「日報表」分頁，欄位：日期、五大KPI、三件大事、連結人數、感謝/優化、明天要做。

⌨ 快捷鍵

🧩 指令清單（白名單 / 黑名單）

      
可限制系統可以執行的指令。選擇模式後勾選要允許的指令。

      

        
        
        
        
      

      

    


    
    

      
🗂 Obsidian 連結

      
設定 Obsidian Vault 目錄路徑，Agent 可調用相關筆記擴充人脈資料。

      

        
        
      

      

    


    
    

      
⏱ Obsidian 定時備份

      
每 5 分鐘自動寫入今日備份到指定資料夾。每天一個檔案（CRM-YYYY-MM-DD.json），可逆向還原任意日期資料。

      

        
        
      

      
未設定自動備份

      
僅支援 Chrome / Edge。授權後每次開啟頁面會自動申請續期。

    


    
    

      
☁️ 雲端同步 & MCP 接入

      
設定 API Token 後，CRM 資料會自動同步到雲端，Hermes 等 AI Agent 可透過 MCP 讀寫資料。

      

        
        
        
      

      

      

        MCP 端點：https://fdd-crm.pages.dev/api/mcp
         ·  首次使用請先呼叫 crm_set_token 工具初始化 Token
      

    


    
    

      
💾 備份與還原
