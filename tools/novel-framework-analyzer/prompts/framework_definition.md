# 局心欲變：正式框架定義 v2.0

## 名稱
**局心欲變**（固定，不可混用其他近似名稱）

## 分析順序
**局 → 心 → 欲 → 變**（固定，不可顛倒）

---

## 局（Situation）— 外部賽局
一個場景中的外部結構與互動條件。

分析點：
- 權力關係：誰強誰弱，強弱如何形成
- 資源分布：誰掌握什麼，稀缺在哪裡
- 主動 / 被動：誰在推動，誰在應對
- 風險與代價：若不行動會發生什麼
- 限制與規則：什麼不能做，什麼被強制
- 籌碼：誰有退路，誰沒有

局不是背景介紹，而是「這場戲的驅動結構」。

---

## 心（Mind）— 認知框架
角色在進入場景時的心態與認知底色。

分析點：
- 初始心態：情緒基調、信念前提、認知框架
- 盲區：角色看不見或不願看見的部分
- System 1 / System 2：直覺反應 vs 理性評估
- 穩定性：容易被動搖 vs 固守

**心不是單純情緒。** 只有情緒波動但沒有認知結構，不能高判為「心」。

---

## 欲（Desire）— 動機結構
角色真正想要的東西，必須區分多層。

層次：
- 顯性欲望：說出來的、做出來的
- 隱性欲望：沒說出口，可能本人未意識到
- 真欲望：驅動所有行動的根本動力
- 偽裝欲望：給別人看的
- 欲望衝突：自我矛盾或與他人碰撞

**欲不等於口頭說法。** 必須區分「嘴上說要什麼」與「實際上要什麼」。

---

## 變（Change）— 轉化結果
場景帶來的可辨識前後差。

必須具備：
1. 前狀態（before）
2. 觸發點（trigger）：具體事件或對話
3. 後狀態（after）
4. 原文證據（直接引用）

轉變類型（shift_type）：
- `emotion`：情緒轉折
- `stance`：立場轉折
- `strategy`：策略轉折
- `values`：價值觀轉折
- `identity`：身份認知轉折
- `none`：無可辨識轉變

轉變強度（shift_intensity）：1–5
- 與類型完全分離，不可混用
- 不能因為是 identity 就自動給 5 分

**沒有前後差不算變。** 短暫情緒波動若無結構性轉折，不算有效「變」。

---

## 框架判定（Match）

評分維度（各 0.0–0.25，合計 0.0–1.00）：
- 局：有清楚外部結構
- 心：有可辨識起點與認知框架
- 欲：顯性 + 隱性欲望可推定
- 變：有原文支持的前後差

Match Level：
- `high`（0.85–1.00）：四維齊備
- `medium`（0.65–0.84）：三維清晰
- `low`（0.40–0.64）：兩維勉強
- `none`（0.00–0.39）：資料不足或不符
