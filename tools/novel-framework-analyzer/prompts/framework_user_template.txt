## 輸出規則
1. 只輸出 JSON，不加任何說明文字或 markdown code fence
2. quotes 中的字串必須直接複製原文，不得改寫或虛構
3. quotes 至少一條，建議 2–4 條覆蓋四個維度的關鍵觸發點
4. situation / mind / desire / change 均為純文字，請用 1–3 句話清楚描述
5. change_intensity 為 1（微弱）至 5（劇烈相變）的整數
6. is_negotiation 為 true 的條件：focal_character 是談判/博弈的主動方，有明確的條件交換、試探、借勢、施壓、讓步行為
7. negotiation_pattern_tags 只填 focal_character 本人的行為模式
   可選標籤：borrowed_pressure / let_opponent_speak_first / exchange_probe / condition_redefinition / risk_transfer / fake_concession / weakness_to_initiative / information_for_space / bluff_detection / tempo_control
